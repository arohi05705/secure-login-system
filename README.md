# Secure Login System

## Description
This is a Python-based Secure Login System that demonstrates brute-force attack prevention.

## Features
- Multiple user login
- Hidden password using getpass
- Maximum 3 login attempts
- 5-second account lock
- Failed login logging
- User authentication using users.txt

## Technologies Used
- Python
- File Handling
- getpass
- time

## Files
- login.py
- users.txt
- login_log.txt

## How to Run

1. Install Python.
2. Open the project folder.
3. Run:

```bash
python login.py
```
